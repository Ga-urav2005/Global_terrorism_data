# Global_terrorism_data
📊 Global Terrorism Data Analysis
📌 Project Overview

This project analyzes the Global Terrorism Dataset to uncover patterns, trends, and insights related to terrorist activities across different regions and time periods. The analysis includes data cleaning, exploration, and visualization to better understand factors like attack frequency, casualties, and temporal trends.

🎯 Objectives
Perform data cleaning and preprocessing
Analyze trends over time (year-wise attacks)
Study casualties (kills & wounds)
Identify high-risk regions and countries
Create visualizations for better insights
📂 Dataset
Dataset used: Global Terrorism Database (GTD)
Features include:
Year of attack (iyear)
Number of kills (nkill)
Number of wounded (nwound)
Country, region, attack type, etc.
🛠️ Technologies Used
Python 🐍
Pandas (Data Manipulation)
NumPy (Numerical Analysis)
Matplotlib & Seaborn (Visualization)
Jupyter Notebook
📊 Data Analysis & Visualization

Some key visualizations used in this project:

📈 Line Charts → Trend of attacks over years
📊 Bar Charts → Country-wise attack comparison
🎻 Violin Plot → Distribution of kills
🔗 Pair Plot → Relationship between variables (kills, wounds, year)
🔍 Key Insights
Certain years show significant spikes in attacks
A few regions contribute to majority of incidents
Casualties distribution is highly skewed (many small events, few extreme ones)
Strong relationship observed between kills and wounds
⚙️ Data Wrangling Steps
Handled missing values
Removed irrelevant columns
Converted data types where required
Filtered important features for analysis
